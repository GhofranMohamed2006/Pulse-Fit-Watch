# Pulse-Fit-Watch
A modern and responsive landing page for a fictional smartwatch brand built using HTML, CSS, and JavaScript.

## Features
- Responsive design for desktop and mobile.
- Modern dark UI.
- Smooth scrolling navigation.
- Contact form validation.
- Interactive pricing section.
- Contact information section.
- Social media links.
## Technologies Used
- HTML5.
- CSS3.
- JavaScript.
